HTML5

HTML5 is a markup language used for structuring and presenting content on the web. HTML5 delivers rich content, such as graphics and videos without plugins. Better semantic support for web structure with new structural element tags. Provides a stricter parsing to simplify error handling.

CSS3

CSS3 is the latest version of CSS. CSS3 provides javascript and mobile development features with additional features such as transitions, gradients, and more support in responsive design. This page is mobile friendly thanks to flexbox and transition features in CSS3.
